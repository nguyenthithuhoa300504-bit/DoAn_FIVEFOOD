# AI Rules

This file contains rules and guidelines for AI coding assistants working on this project.
QUY TẮC BẮT BUỘC

Đây là dự án đã hoàn thiện toàn bộ chức năng. Mục tiêu chỉ là cải thiện giao diện (UI/UX).

TUYỆT ĐỐI KHÔNG ĐƯỢC:

- Thiết kế lại toàn bộ website.
- Thay đổi bố cục tổng thể.
- Xóa, thay thế hoặc đổi bất kỳ hình ảnh nào.
- Thay đổi Hero Banner hoặc Slider hiện có.
- Thay đổi dữ liệu hiển thị.
- Thay đổi API.
- Thay đổi Backend.
- Thay đổi Database.
- Thay đổi Route.
- Đổi tên Component.
- Xóa Component.
- Tạo Component mới nếu không thật sự cần.
- Thay đổi logic nghiệp vụ.
- Thay đổi cấu trúc thư mục.
- Cài đặt thêm thư viện nếu chưa được sự đồng ý.

CHỈ ĐƯỢC PHÉP:

- Cải thiện UI.
- Cải thiện CSS.
- Cải thiện khoảng trắng.
- Cải thiện Typography.
- Cải thiện màu sắc.
- Cải thiện Responsive.
- Cải thiện Animation.
- Cải thiện trải nghiệm người dùng.
- Tối ưu hiệu ứng nhưng không làm thay đổi chức năng.

Nếu cần chỉnh JSX thì:
- Giữ nguyên cấu trúc component.
- Giữ nguyên chức năng.
- Chỉ chỉnh khi thật sự cần thiết.

Sau khi hoàn thành:
1. Liệt kê tất cả các file đã chỉnh sửa.
2. Giải thích ngắn gọn lý do chỉnh sửa từng file.
3. Không được sửa các file không liên quan đến yêu cầu hiện tại.
4. Nếu phát hiện cần sửa ngoài phạm vi yêu cầu thì phải hỏi lại trước khi thực hiện.
Chỉ cải thiện giao diện Header.

Giữ nguyên:
- Logo.
- Thanh tìm kiếm.
- Các nút hiện có.
- Chức năng hiện có.

Chỉ điều chỉnh:
- Khoảng trắng.
- Căn lề.
- Kích thước.
- Màu sắc.
- Hover.
- Shadow.
- Responsive.

Không thay đổi JSX nếu không cần.

Không sửa component khác.
